# CsvToJsonl
Print CSV as JSONL format

```
usage: CsvToJsonl.py [-h] -f CSV_FILE -d DELIMITER

CSV to JSONL

optional arguments:
  -h, --help            show this help message and exit
  -f CSV_FILE, --file CSV_FILE
                        The csv file to transform
  -d DELIMITER, --delimiter DELIMITER
                        The field delimiter
```
